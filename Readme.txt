Програмко:
Стартует с андроидом подписывается на получение местоположения.
Один раз в минуту запускается новый поток требующий местоположения,
в результате, в случае штатной работы GPS в течении минуты должен быть фикс.
при старте выполняет сброс AGPS


При отсутствии проверочного фикса всплывает окно:

чекбоксики
- Показывать окно при проблемах GPS
если установлен при проблеме запуститься данное окно,
при выключенном потоки будут работать, но окно не будет беспокоить
-Закрыть окно после нажатия на кнопку 'Очистка'
если установлено - при нажатии на кнопку очистка - окно закроется
Кнопки:
собственно то, что на них написано.

Предполагаемое использование:
программа ставиться, при следующем запуске потоки стартуют с андроидом
при появлении окна бросаем взгляд  на скорость координаты
(если не меняются)  - мы повисли, жмакаем очистить AGPS - окно закроется.(см чекбоксики)
(если меняются)  - я лошара - ложное срабатывание жмакаем выход - окно закроется.
